# DJ Artwork Generator

## Usage

1. Add your `template.psd` to `public/template.psd` (replace the placeholder).
2. Ensure your fonts are in `public/fonts/`.
3. Deploy the `public/` folder to Vercel or serve locally.

## Files

```
my-dj-artwork/
├── public/
│   ├── index.html
│   ├── template.psd      # placeholder—replace with your PSD
│   └── fonts/
│       ├── NeueHaasDisplayThin.ttf
│       └── ...
├── vercel.json
└── README.md
```
